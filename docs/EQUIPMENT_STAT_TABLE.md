# Equipment Stat Table

Each subtype has a stat pool with value ranges per tier.
A specific item (icon) receives **5 stats** from the subtype pool (hash of the filename).
On drop, 2-6 of those 5 are randomly selected depending on rarity.

**Tiers:** T5 (iLvl 1-19) → T4 (20-39) → T3 (40-59) → T2 (60-79) → T1 (80-100)
